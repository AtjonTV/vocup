# Vocup file formats

Vocup uses different proprietary formats:

| Extension | Name                            | Usage            |
|-----------|---------------------------------|------------------|
| .vhf      | **V**okabel**h**eft **f**ile    | Vocabulary data  |
| .vhr      | **V**okabel**h**eft **r**esults | Practice results |
